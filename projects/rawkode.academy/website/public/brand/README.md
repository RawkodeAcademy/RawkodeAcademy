Brand assets and downloadable files for Rawkode Academy.

Place any public downloadable brand assets here. Our website intentionally does not include the full Brand Guidelines PDF; partners should request the kit through our partnership contact.

Accepted assets:
- `monogram.svg`
- `wordmark.svg`
- `logo.svg`

All vector logos must be black-only in the repository; our website uses CSS to theme them for light/dark/colored contexts.

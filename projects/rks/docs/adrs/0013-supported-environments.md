## Decision
- Producer: Chrome desktop only.
- Guests: Chrome desktop only.
- Viewers: Chromium/Safari/Firefox on desktop; iOS Safari and Android Chrome on mobile.

## Consequences
- Focused QA; fastest path to v1.
